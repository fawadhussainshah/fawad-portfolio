# Public Assets

## Profile Photo

Currently using a placeholder image from Unsplash. To use your own photo:

1. Save your professional photo as `profile-photo.jpg` in this folder
2. Recommended size: 400x400 pixels or larger (square aspect ratio)
3. Update `src/components/Hero.jsx` line 35 to use `/profile-photo.jpg` instead of the Unsplash URL
4. The image will be automatically cropped to a circle in the portfolio

Supported formats: JPG, PNG, WebP

**Note:** The portfolio currently uses a professional placeholder image. Replace it with your actual photo for a personalized touch!
